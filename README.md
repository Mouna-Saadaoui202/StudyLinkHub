# StudyLinkHub

Vite + React website for Study Link Hub.

## Active Website Code

The current landing page and blogs live in:

```txt
src/features/site/
```

- `pages/`: active landing and blog pages.
- `data/siteContent.js`: bilingual content, contact details, social links, and blog data.
- `styles/landing.css`: active website styles.

Older ZIP template folders are still present under `src/components`, `src/layouts`, and older `src/pages/*` folders. They are backup/reference files, not the current routed website.

## Commands

```bash
npm install
npm run dev
npm run build
```
