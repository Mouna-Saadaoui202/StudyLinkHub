# StudyLinkHub active site

This folder contains the current StudyLinkHub website that is wired in `src/App.jsx`.

- `pages/` contains the active landing page and blog pages.
- `data/siteContent.js` contains bilingual text, blog content, contact details, and asset references.
- `styles/landing.css` contains the active website styles.

The older ZIP template folders still exist under `src/components`, `src/layouts`, and older `src/pages/*` folders. They are kept only as a backup/reference and are not part of the current routed website.
